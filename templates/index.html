WHAT IS A BLOCKCHAIN?

Blockchain is a growing technology that allows for the storage and sharing of data in a way that is verifiable, immutable, and decentralized.
The blockchain is a distributed ledger of information, which is stored in a public database, represented as blocks, which are connected to each other by hashes; each block contains a cryptographic hash of the previous block, making the state of the Blockchain immutable.

WHAT MAKES THE IC–BLOCKCHAIN SPECIAL?

The Internet Computer is a layer, built ontop of the Internet Protocol, called the Internet Computer Protocol (ICP), representing the chain.
The Internet Computer Protocol offers many advantages over other blockchains. These include:

  • The scalability, accessibility and security to date are unprecedented in the market
  • Unlike competing platforms, the end user does not pay for computing services on the Internet computer
  • Users do not need to own cryptocurrency to interact with Internet Computer Protocol
  • The service is designed in such a way that users would not notice or have to adapt to the decentralized technology on which ICP is based on. The transition for the users is as seamless as possible
